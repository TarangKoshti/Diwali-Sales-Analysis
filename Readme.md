# 🪔 Diwali Sales Data Analysis

## 📌 Project Overview
This project focuses on analyzing customer purchasing behavior during the Diwali festival using Python. The objective is to uncover meaningful insights that can help businesses make data-driven decisions, improve customer targeting, and increase overall sales.

---

## 🎯 Problem Statement
To analyze Diwali sales data and identify key patterns in customer demographics, purchasing behavior, and regional performance.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset Description
The dataset contains transactional data of customers during Diwali sales, including:
- Gender
- Age Group
- Marital Status
- State
- Occupation
- Product Category
- Purchase Amount

---

## 🧹 Data Cleaning & Preparation
- Removed unnecessary columns (`Status`, `Unnamed: 1`)
- Handled missing values by dropping null entries
- Converted data types for accurate analysis
- Verified data consistency

---

## 📊 Exploratory Data Analysis (EDA)
The following analysis was performed:
- Gender-wise customer distribution
- Sales comparison based on gender
- Age group analysis
- Top 10 product categories
- State-wise sales performance
- Occupation-based purchasing trends

---

## 🔍 Key Insights

- Female customers are the primary contributors to sales, both in count and total spending
- The age group 26–35 represents the most active and high-value customers
- A small number of product categories dominate overall sales
- Sales are concentrated in a few high-performing states
- Customers from professional sectors such as IT and Healthcare tend to spend more

---

## 💡 Business Recommendations

- Target female customers with personalized marketing campaigns
- Focus on the 26–35 age group for maximum conversions
- Prioritize inventory and promotions for top-selling product categories
- Strengthen marketing and logistics in high-performing states
- Offer premium deals to working professionals

---

## ✅ Conclusion

The analysis highlights clear patterns in customer behavior, showing that targeted marketing based on gender, age group, and region can significantly boost sales performance. Businesses can leverage these insights to optimize their strategies and improve customer engagement.
